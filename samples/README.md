# Demo sample images

The three `*_demo.jpg` files in this folder are **synthetic placeholders** (simple color patches generated in-repo) so the Streamlit **Mock** backend works immediately after clone without downloading HAM10000.

For publication-grade demos, replace them with real dermoscopic images **only if you have rights to redistribute** (for example, images you captured yourself, or HAM10000-derived crops used in compliance with the [HAM10000 Scientific Data terms](https://doi.org/10.1038/sdata.2018.161) and your local ethics rules). Document image IDs and license in this README when you swap them in.
